# Mini C Compiler (Python)
A small C subset compiler that lexes, parses, and generates LLVM IR, then
compiles it down to a native executable.

## Usage
```
python main.py <filename>.c
```
This produces an executable named `a` in the current directory.

## Progress
Done:
- Lexical analysis
- Parser
- IR Codegen
- printf/scanf, string literals

See `grammar.md` for the supported grammar.
